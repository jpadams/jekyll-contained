# `jekyll-contained`

A contained version of Jekyll.

![jekyll-contained](https://user-images.githubusercontent.com/27806/67113157-6c8fc900-f19e-11e9-92ab-4106ef5ad22d.png)

## Usage

```shell
# First setup the site
./script/setup.sh

# Then build the site
./script/build.sh

# Now serve the site
./script/serve.sh
```
